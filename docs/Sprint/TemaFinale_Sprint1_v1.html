Goal

Realizzare il sottosistema minimo che gestisce l'accettazione/rifiuto di una richiesta di carico e la parte relativa al timeout dei 30 secondi. cargoservice riceve loadrequest, interroga Hold, risponde con loadaccepted/loadrejected, e passa da Idle a Engaged; se il timeout scade torna a Idle liberando lo slot.

Problem Analysis

Architettura logica del sottosistema

Rispetto al modello qak di Sprint0, in questo sprint restano solo l'attore cargoservice e gli oggetti Java che usa internamente (Hold, IOPortStatus). In questa fase del progetto, come già detto, ci occuperemo della logica di business e, pertanto, abbiamo scelto di rappresentare una classe IOPortStatus mock che simuli il comportamento dell'IOPort che andremo a sviluppare in futuro.

QActor cargoservice context ctxcargoservice
    withobj hold using "utils.cargoservice.Hold()"
    withobj ioport using "utils.cargoservice.IOPortStatus()" {

    [# var Slot = -1 #]

    State idle initial {
        println("$name | idle, in attesa di richieste...") color magenta
    }
    Transition t0 whenRequest loadrequest -> handleRequest

    State handleRequest {
        onMsg(loadrequest : loadrequest(X)) {
            if [# ioport.isOccupied() #] {
                replyTo loadrequest with loadrejected : loadrejected(retrylater)
            } else {
                [# Slot = hold.reserveFirstFree() #]
                if [# Slot != -1 #] {
                    forward cargoservice -m blinkLed : blinkLed(on)
                    replyTo loadrequest with loadaccepted : loadaccepted($Slot)
                } else {
                    replyTo loadrequest with loadrejected : loadrejected(full)
                }
            }
        }
    }
    Goto engaged if [# Slot != -1 #] else idle

    State engaged {
        println("$name | engaged") color magenta
    }
    Transition t1 whenTime 30000 -> disengaged

    State disengaged {
        println("$name | timeout scaduto, libero lo slot e torno idle") color magenta
		forward cargoservice -m blinkLed : blinkLed(off)        
		[# hold.releaseReserved() #]
        [# Slot = -1 #]
    }
    Goto idle
}

Modifiche alla classe Hold

Rispetto alla Hold dello Sprint0, sono state apportate delle modifiche:

  • E' stato modificato il metodo reserveFirstFree(). Per esigenze legate a semplicità, abbiamo scelto di cambiare il suo tipo, passando da String a int. A ogni slot è quindi associato un numero intero da 1 a 4 e il null è stato sostituito con un -1 simbolico.
  • Nella Hold di Sprint0 non era previsto liberare uno slot RESERVED. Serve però per gestire R2: se il cliente non porta il container entro un tempo prefissato (in questo caso 30s), lo slot prenotato deve tornare libero e lo stato disengaged, altrimenti la hold si "intaserebbe" di prenotazioni fantasma.
public class Hold {
    public static final int NUM_SLOTS = 4;       // slot1..slot4

    public enum SlotState { FREE, RESERVED, OCCUPIED }

    private SlotState[] slots = {
        SlotState.FREE,
        SlotState.FREE,
        SlotState.FREE,
        SlotState.FREE
    };

    private boolean slot5Occupied = false;

    public boolean isSlot5Occupied() {
        return slot5Occupied;
    }

    public void setSlot5Occupied(boolean occupied) {
        slot5Occupied = occupied;
    }

    public int reserveFirstFree() {
        for (int i = 0; i < NUM_SLOTS; i++) {
            if (slots[i] == SlotState.RESERVED) {
                return -1; // al più una prenotazione attiva alla volta
            }
        }
        for (int i = 0; i < NUM_SLOTS; i++) {
            if (slots[i] == SlotState.FREE) {
                slots[i] = SlotState.RESERVED;
                return (i + 1);
            }
        }
        return -1; // piena
    }

    public void markReservedOccupied() {
        for (int i = 0; i < NUM_SLOTS; i++) {
            if (slots[i] == SlotState.RESERVED) {
                slots[i] = SlotState.OCCUPIED;
                return;
            }
        }
    }

    public void releaseReserved() {
        for (int i = 0; i < NUM_SLOTS; i++) {
            if (slots[i] == SlotState.RESERVED) {
                slots[i] = SlotState.FREE;
                return;
            }
        }
    }

    public boolean isFull() {
        for (SlotState s : slots) {
            if (s != SlotState.OCCUPIED) {
                return false;
            }
        }
        return true;
    }

    public String displayStatus() {
        int occupied = 0;
        for (SlotState s : slots) {
            if (s == SlotState.OCCUPIED) {
                occupied++;
            }
        }
        return occupied + "/" + NUM_SLOTS + " occupied";
    }
}

Test Plan

T1.6 Dopo reserveFirstFree(), una chiamata a releaseReserved() deve far tornare libero lo slot: una successiva reserveFirstFree() deve restituire lo stesso slot.

Riportiamo qui sotto la classe HoldTest modificata, in quanto sono stati modificati alcuni metodi nella classe Hold.

class HoldTest {

    Hold hold;

    @BeforeEach
    void setUp() { hold = new Hold(); }

    @Test // T1.1
    void reserveFromEmpty() {
        assertEquals(1, hold.reserveFirstFree());
    }

    @Test // T1.2
    void fullWhenAllOccupied() {
        for (int i = 0; i < Hold.NUM_SLOTS; i++) {
            hold.reserveFirstFree();
            hold.markReservedOccupied();
        }
        assertTrue(hold.isFull());
    }

    @Test // T1.3
    void reserveWhenFull() {
        for (int i = 0; i < Hold.NUM_SLOTS; i++) {
            hold.reserveFirstFree();
            hold.markReservedOccupied();
        }
        assertEquals(-1, hold.reserveFirstFree());
    }

    @Test // T1.4
    void markReservedBecomesOccupied() {
        hold.reserveFirstFree();
        hold.markReservedOccupied();
        assertFalse(hold.isFull()); // 1/4, non piena
        assertNotEquals(-1, hold.reserveFirstFree());
    }

    @Test // T1.5
    void noDoubleReserve() {
        hold.reserveFirstFree(); // slot1 → RESERVED
        assertEquals(-1, hold.reserveFirstFree());
    }

    @Test // T1.6 
    void releaseReservedFreesTheSlot() {
        hold.reserveFirstFree(); // slot1 -> RESERVED
        hold.releaseReserved();
        assertEquals(1, hold.reserveFirstFree());
    }
}
